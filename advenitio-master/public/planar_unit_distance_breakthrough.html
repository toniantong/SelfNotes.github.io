簡介與核心發現

近 80 年來，數學界一直研究著一條表面上非常簡單的組合幾何問題：如果你在平面上面放置 n 個點，當中有多少對點的距離可以剛好是 1？

這個著名的問題被稱為「平面單位距離問題」（planar unit distance problem），最早由傳奇數學家保羅·艾狄胥（Paul Erdős）於 1946 年提出。幾十年來，無數頂尖幾何學家投身其中卻未能取得本質突破。今日，OpenAI 的一個全新通用推理模型成功打破了這個長久以來的僵局，實現了多項式級別（polynomial）的改進，並首次由 AI 自主解開了處於數學核心分支的著名未解難題。

菲爾茲獎得主 Tim Gowers 形容這個結果是「AI 數學領域的一個里程碑」。此證明並非來自一個專門為數學訓練或搜索證明策略設計的系統，而是出自一個全新通用推理模型深度推理的成果。

單位距離問題背景

設 u(n) 為平面上 n 點之間可以形成最多單位距離對的數量。要構造出達到線性增長率的例子非常容易（例如排成直線可得 n-1 對，正方形網格可得約 2n 對）。

在本次突玻之前，已知最理想的構造方法來自「縮放過的正方形網格」，其結果為：

u(n) ≥ n1 + C / log log(n)   (C 為常數)

隨著 n 趨向無限大，指數中的附加項會趨向 0。因此幾十年來學界普遍猜想艾狄胥的上限是 n1+o(1)（其中 o(1) 代表隨 n 趨向無窮而消失的項），認為沒有結構能大幅超越正方形網格。

新結果推翻了這一猜想：證明構造了至少有 n1+δ 個單位距離對的配置，其中 δ > 0 是固定的常數。普林斯頓大學教授 Will Sawin 隨後將其改良並明確指出可以取 δ = 0.014。

回顧歷史，最理想的下限自 1946 年以來基本停滯；而最理想的上限 O(n4/3) 則一直停留在 1984 年 Spencer、Szemerédi 和 Trotter 的研究。本次發現直接在兩者之間開闢了全新的多項式級別改進空間。

跨領域的全新技術：代數數論

令人意想不到的是，解決這個純幾何學問題的關鍵元素竟然來自於數學的另一個截然不同的分支 —— 代數數論（algebraic number theory）。

艾狄胥最初的下限可以透過高斯整數（Gaussian integers，即形式為 a+bi 的數字）來理解。高斯整數是普通整數的擴展，具有質因數唯一分解等特性。新證明的核心策略是將高斯整數替換成代數數論中更為複雜的推廣，利用更豐富的對稱性，在空間中創造出多得多的單位長度差。

具體論證中，模型精妙地運用了無限類體塔（infinite class field towers）以及 Golod-Shafarevich 理論等高深工具，去證明所需的數體確實存在。這些在數論學家眼中熟悉的工具，跨界應用到歐幾里得平面的幾何問題上，帶來了驚人的突破。

長遠意義與科研未來

這項成果的啟示遠超於單一幾何學難題的解決。它揭示了 AI 作為「科學研究夥伴」的全新可能：

  • 跨領域概念連結：模型展現了能夠跨越遙遠知識領域（幾何學與高深數論）並將其融合的深度推理能力。
  • 拓展科研邊界：AI 能夠維繫極其複雜的思維鏈條，發掘人類專家可能忽略的潛在路徑。
  • 廣泛的科學應用：這種長篇邏輯推理與跨領域整合能力，未來同樣可應用於生物學、物理學、材料科學、工程學及醫學等領域。

這代表著我們邁向自動化科學研究（AI for Science）的重要里程碑。AI 即將在研究的創意部分扮演非常認真的角色，而人類的專業知識將變得更加有價值 —— 選擇重要問題、合理解釋結果、並決定下一步探究方向，始終是人類的責任。
